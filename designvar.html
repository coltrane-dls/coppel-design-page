Variables de diseño

Este documento contiene variables de colores como primarios de interacción, textos, backgrounds, borders. Ademas de tamaños tipografías, curvatura de bordes y sombras de profundidad.

Variables de diseño
También conocido como "tokens de diseño"

Estas son una única fuente de verdad para nombrar y almacenar decisiones de diseño para experiencias de productos Coppel.

¿Qué son y para qué sirven?

Las variables de diseño son combinaciones de nombre y valor que representan decisiones de diseño pequeñas y reutilizables. Una variable puede ser un color, un estilo de fuente, una unidad de espacio en blanco o profundidad, diseñada para una necesidad específica.

Estos elementos pueden ser consumidos por desarrolladores y diseñadores dentro de Grupo Coppel.
